# Ищи query-in-loop и performance hotspots

- Запросы внутри циклов и неявные запросы через `Ссылка.Реквизит` в массовой
  обработке — основной источник деградации. Помечай и требуй переписать на
  пакетное получение данных.
- Проверяй работу с большими выборками: выгрузка всего в память, отсутствие
  отборов/индексов, лишние полные обходы.
- Смотри на транзакции и блокировки: вложенные транзакции в обработчиках
  записи/проведения, избыточные/неуправляемые блокировки.
