# Сверь изменения с моделью метаданных

- Проверь, что все используемые имена объектов, реквизитов и типов реально
  существуют в метаданных.
- Убедись, что правки модулей согласованы со структурой форм, табличных
  частей и движений регистров.
- Для расширений проверь корректность заимствований и точек перехвата,
  отсутствие лишней связности с типовой.
