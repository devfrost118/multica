# Финальный отчёт с доказательствами

Завершая задачу, дай отчёт, по которому изменение можно проверить:

- список изменённых файлов с путями;
- что именно изменено и почему (привязка к задаче);
- какие проверки реально запускались и их результат/код возврата;
- известные непроверенные зоны и риски;
- заметки для ревью/runtime-теста в 1С и заметки по откату.

Не выдавай предположение за проверенный факт. Если проверить не удалось —
так и скажи.
