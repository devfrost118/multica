# Найди точный объект перед правкой

Прежде чем писать код:

- Определи конкретный объект: справочник/документ/регистр/обработка, его форму,
  модуль и при необходимости движения.
- Для перехвата в расширении (CFE) найди заимствованный объект и точку
  перехвата (`&Перед`, `&После`, `&Вместо`), а не дублируй логику.
- Убедись, что правишь именно тот модуль, где живёт нужная логика, а не
  одноимённый метод в другом месте.
- Если объект не найден — остановись и уточни, не создавай «похожий» наугад.
